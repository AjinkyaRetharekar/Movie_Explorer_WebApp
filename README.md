#  Movie Explorer Web App

A modern movie discovery web application built with **React (Vite)**, powered by the **TMDB API** for real-time movie data, and integrated with **Appwrite Database** to track trending searches.


# About the Project

This app allows users to search and explore movies in real-time.  
It fetches movie details from **The Movie Database (TMDB) API**, while **Appwrite** stores and ranks trending movie searches based on popularity.

Key features include:
-  Real-time movie search with debouncing  
-  “Trending Movies” section using Appwrite’s database  
-  Dynamic poster handling (fallback for missing posters)  
-  Fast performance via Vite bundler  

---

###  Tech Stack

| Frontend -> React (Vite), JavaScript (ES6), CSS |
| API -> TMDB (The Movie Database API) |
| Backend -> Appwrite Cloud Database |
